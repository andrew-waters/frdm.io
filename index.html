frdm

An open source Content Management API

        aw ~ $

Fast

Written in Go, with internal communicatin over gRPC, frdm is a performant collection of microservices.

Well Tested

Each microservice contains extensive test suites and the API has it's own acceptance tests.

Open Source

Simply grab the code and deploy it. It's a doddle. See a microservice you want to use for another project. Just grab it!

Configuration

Maintain all your settings in a single ENV file. Deploy to multiple environments with multiple ENV files.

Extendable

Want a new service to do something new? Write it, deploy it and add it to the gateway and you're away.

Heterogenous

Data is sent by default as JSON, making it a breeze to consume the API via any language.
